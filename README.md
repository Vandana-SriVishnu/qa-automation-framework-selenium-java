# Hybrid QA Automation Framework – Selenium + Java

## Overview
This repository demonstrates a hybrid QA approach combining manual testing, UI automation using Selenium (Java), and API testing using Postman. It reflects a senior-level QA mindset suitable for enterprise and remote projects.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Postman
- GitHub Actions (CI/CD – upcoming)

## Framework Highlights
- Manual test case design
- Page Object Model (POM)
- UI automation for core user flows
- API testing for backend validation
- Scalable and maintainable framework structure

## Test Application
https://www.saucedemo.com/
## CI/CD Integration
This project integrates Selenium automation with GitHub Actions. Tests are executed automatically in a headless Chrome environment on every push and pull request, ensuring continuous quality validation in a remote CI setup.
